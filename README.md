Текстовый редактор с тремя доступными тарифами (FREE / BASIC / VIP) и встроенным графическим редактором (растровое рисование + векторные фигуры). Данные пользователей, подписок, избранного, истории и заметок хранятся в облачной базе данных Supabase.
Функциональность

Текстовый редактор
- Автосохранение каждые 30 секунд с восстановлением после сбоев
- Открытие и сохранение файлов в форматах `.html` и `.docx` (Apache POI)
- Поиск и замена текста с учётом специальных символов
- Статистика документа (символы, слова, абзацы)
- Проверка орфографии через HTTP‑API LanguageTool
- Выбор шрифтов с ограничением по подписке (3 / 10 / 30)
- Расширенное форматирование (жирный, курсив, подчёркнутый, выравнивание, списки, цвет, размер шрифта) – для платных планов

Графический редактор
- Рисование карандашом и ластиком
- Создание векторных фигур (прямоугольник, круг, эллипс, линия, треугольник)
- Выделение, перемещение и изменение размера фигур (якоря для круга/эллипса)
- Анимация пузырьков при каждом действии
- Сохранение рисунка в PNG
